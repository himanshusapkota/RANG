# 🎨RANG

Rang is a nepali word for a color.
The Rang is a simple web application that generates beautiful random color palettes for the inspiring designer and develpers. Instantly create color combinations, copy HEX codes, and save your favorite palettes for future projects.


## ✨Features
- Generate random color palettes
- copy HEX codes with one click
- Save favorite palettes
- Local Storage Support
- Responsive Design
- Smooth Animation

## Built with
- HTML5
- CSS3
- JavaScript

## Get Started with RANG

1. Clone this repository

```bash
git clone https://github.com/himanshusapkota/RANG.git
```

2. Open `index.html` in your browser
